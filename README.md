# forumbuilder
一个可以用来生成论坛体的东西。
支持多楼层编辑、时间轴自动生成、富文本编辑、自定义模板与一键导出。

在线地址
https://cyphadr.github.io/forumbuilder/

---

- 📌 多楼层帖子编辑（自动编号）
- 📝 富文本编辑（加粗 / 斜体 / 下划线 / 链接）
- ⏰ 自动时间轴生成（支持 12h / 24h / 年份）
- 🆔 随机 ID 生成（8 位字母数字）
- 📄 多种结构模板（Classic / Compact / Bubble / Custom）
- 📤 导出为 HTML / TXT（如果使用AO3 workskin可以使用自定义模式批量生成对应的html）
- 💾 自动本地保存（localStorage）
- 📱 适配手机 / 桌面端使用

---


首次打开会自动弹出【初始化设置】窗口。

---

初始化设置

建议第一次使用先完成设置：

- 选择帖子结构风格
- 勾选显示字段
- 设置时间格式
- 是否开启随机 ID
- 是否使用自定义模板

设置完成后点击：

【应用并进入编辑】

---

编辑帖子内容

在编辑区中：

- 每一层楼对应一个折叠面板
- 可填写：
  - 昵称
  - ID
  - 时间
  - 引用
  - 正文

支持富文本编辑：

- **B** 加粗
- *I* 斜体
- U 下划线
- 链接
- 换行

---

时间轴功能（可选）

支持自动生成时间轴：

1. 输入首楼时间（作为锚点）
2. 开启随机时间
3. 点击【生成并锁定】

系统会自动推算后续楼层时间。

---

 预览与导出

点击：

【生成 / 刷新预览】

即可生成导出内容。

支持：

- 一键复制 HTML
- 一键复制 TXT
- 下载文件

---

## 💾 本地存储说明（重要）

本工具使用浏览器的 `localStorage` 自动保存编辑进度。
仅在以下条件下可读取同一份存档：

- 同一浏览器
- 同一设备
- 同一域名

不同浏览器 / 设备 / 域名之间无法同步数据。

如需迁移或备份，请使用导出 / 导入功能（JSON）。
